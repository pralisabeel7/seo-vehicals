Quick SEO Notes

Short-Tail vs Long-Tail Keywords

Short-Tail: 1-2 words, high competition (e.g., "car modification")

Long-Tail: 3+ words, better conversion (e.g., "fj cruiser dashboard upgrade Pakistan")

Focus on long-tail with vehicle model + service + location

Hashtag Strategy

YouTube: 3-5 relevant hashtags in description

Facebook/Instagram: 5-10 hashtags per post

Mix: Broad + specific + location + branded hashtags

Research trending auto modification hashtags regularly

Essential Tools

Must Use: Google Keyword Planner, Search Console, Analytics

For Competition: Ahrefs, Ubersuggest

For Technical: Screaming Frog, PageSpeed Insights

Check tools weekly for updates

Local SEO Focus

Location Keywords: "auto shop Lahore", "car modification Karachi"

Google My Business: Optimize completely, add photos

Local Listings: Ensure NAP consistency across directories

Create location-specific pages if serving multiple cities
